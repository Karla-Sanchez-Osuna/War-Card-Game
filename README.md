
**War Card Game**

This project is an automated version of the classic card game WAR, implemented using Java classes and objects.

**Game Description**

WAR is a simple card game where two players compete to see who has the highest card. The game continues until one player has won all the cards.

**Features**

* Implemented using Java classes and objects
* Automated gameplay with random card dealing
* Keeps track of the number of rounds and the winner

**Technologies Used**

* Java
* JUnit (for unit testing)

**How to Run**

1. Clone the repository to your local machine.
2. Open the project in your preferred IDE.
3. Compile and run the `WarCardGame` class.

**How to Play**

1. The game will automatically deal two cards, one for each player.
2. The player with the higher card wins the round.
3. The game continues until one player has won all the cards.
4. The winner is announced at the end of the game.
